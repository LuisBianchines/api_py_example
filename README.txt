-- instalar venv
	Para criar um ambiente virtual, execute o seguinte comando:
		python -m venv nome_do_ambiente
	
	Substitua nome_do_ambiente pelo nome que você deseja dar ao seu ambiente virtual. Por exemplo:
		python -m venv myenv

	Isso criará um novo diretório com o nome especificado contendo o ambiente virtual.

	Para ativar o ambiente virtual, navegue até o diretório Scripts dentro do ambiente virtual e execute o script activate.bat. 	Você pode fazer isso com o seguinte comando:
		caminho\para\o\diretorio\nome_do_ambiente\Scripts\activate
